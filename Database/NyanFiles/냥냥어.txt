냥습/A/B (A, B는 문자열)
- 상대방이 A를 말하면 화냥봇이 B를 말하도록 학습시킵니다.

냥습/A (A는 문자열)
- 학습데이터 A가 있는지 확인합니다.

삭제/A (A는 문자열)
- 학습데이터 A를 삭제합니다.

말/A (0 <= A <= N(N은 최대 500), A는 정수)
- A번째 이전에 어떤 사람이 무슨 말을 했는지 확인합니다.

화냥폰/A (A ∈ { 버전, 배터리, 전압, 온도, 충전중? })
- 화냥폰의 정보를 봅니다.

순위/A (A ∈ { 말, 사진, 임티, 화냥봇, L, 꿀꺽, 퉤엣, 도망, 꿀꺽당햇, 퉤엣당햇, 꿀꺽대결 })
- 순위를 봅니다.

가바보/A (A ∈ { 가위, 바위, 보 })
가위바위보/A (A ∈ { 가위, 바위, 보 })
- 가위바위보를 합니다.
