냥습/A/B (A, B는 문자열)
- 상대방이 A를 말하면 화냥봇이 B를 말하도록 학습시킵니다.

냥습/A (A는 문자열)
- 학습데이터 A가 있는지 확인합니다.

삭제/A (A는 문자열)
- 학습데이터 A를 삭제합니다.

말/A (0 <= A <= N(N은 최대 500), A는 정수)
- A번째 이전에 어떤 사람이 무슨 말을 했는지 확인합니다.

화냥폰/A (A ∈ { 버전, 배터리, 전압, 온도, 충전중? })
- 화냥폰의 정보를 봅니다.

순위/A (A ∈ { 말, 사진, 임티, 화냥봇, L, 꿀꺽, 퉤엣, 도망, 꿀꺽당햇, 퉤엣당햇, 꿀꺽대결 })
- 순위를 봅니다.

가바보/A (A ∈ { 가위, 바위, 보 })
가위바위보/A (A ∈ { 가위, 바위, 보 })
- 가위바위보를 합니다.

냥냥어
- 냥냥어 도움말을 봅니다.

냥습목록
- 학습 목록을 봅니다.

오늘은
- 오늘 날짜를 봅니다.

요일은
- 오늘 요일을 봅니다.

안녕
안녕하세요
- 인사를 합니다.

화냥봇
- 화냥봇을 부릅니다.

너의 이름은
- 화냥봇의 이름을 봅니다. (당연히 화냥봇이겠지만...)

내 이름은
- 당신의 이름을 봅니다.

꿀꺽
- 가장 최근에 말한 사람(봇은 제외)을 꿀꺽합니다. (단, 1/3 확률로 꿀꺽에 실패할 수 있음)

퉤엣
퉷
- 가장 최근에 꿀꺽한 사람을 퉤엣합니다.

꿀꺽주머니
- 당신의 꿀꺽주머니를 봅니다.

이모지/A (1 <= A의 길이 <= 5, A는 문자열)
- A를 이모지로 만들어서 보여줍니다.
